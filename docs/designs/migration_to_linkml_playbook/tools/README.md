# tools/

Scripts and probes accumulated across migration attempts: diff helpers, ad-hoc validators, schema comparators, one-off reproducers.

**Convention:** each script starts with a one-line header comment naming its purpose, plus a short usage example. If a tool proves durable, consider whether it belongs upstream (in `tools/linkml_conversion_tools/` at the repo root, or in `pydantic2linkml`) rather than living here.
